Proyecto 1 --Agencia de Modelaje (EDD)
## Integrantes: 
Miguel Gómez, Juan Jose Florez y Samuel Alvarez 
## Introducción: 
El proyecto trata sobre el desarrollo de un sistema en Java que representa el funcionamiento de una agencia de modelaje. 
El sistema permite mostrar y gestionar distintos tipos de eventos organizados por la agencia, asi como las personas que participen en dichos eventos, como por ejemplo los modelos y fotógrafos. 
## Funcionamiento del proyecto: 
El proyecto simula el funcionamiento de una agencia de modelaje que otrganiza diferentes tipos de eventos. Las clases se dividieron en paquetes para garantizar el orden y la correcta estructura de herencias.  
A partir de la clase "Evento" surgen dos tipos especificos lo cuales son "evento publico" y "evento privado". Ademas de esto el sistema permite relacionar a cada evento con las personas que participen en él, como los modelos y fotógrafos. 
## Descarga del proyecto: 
Para utilizar el proyecto se puede clonar el repositorio y ejecutarlo en un entorne de desarrollo como Intellij. De esta manera se podra ver la interfaz en donde se puede empezar a manipular la información de la agencia de modelaje.
